This repository contains a collection of scripts that extract statistics from single molecule traces.

AOT data analysis

1. AOT_replisome_data_analysis.m
Script to load raw traces from the Angular Optical Trap (AOT) and calculate the fork position from the extension. This script plots the trace and can measure the stall torque of each trace.  Last, regions of the data are collected and export for further analysis, below.

2. replication_position_vs_time_analysis.m
Script to process replication traces to extract the mean velocity and pause free velocity of the replisome.

3. calculate_velocity_position.m
Function that is used to calculate the velocity and smoothed position from the position vs time data.  Used in (2)

4. collect_velocity_stats.m
Script to collect and export the velocity stats from .mat files and save a TXT file.

5. average_trace_regression.m
Script to calculate the average regression trace from a collection of single molecule traces of regression.  

6. average_trace_resumption.m
Script to calculate the average resumption trace from a collection of single molecule traces of resumption.

MT data analysis

1. MT_resumption_duration_v5_4.m
Script to process single molecule traces from a Magnetic tweezers experiment.  Measures and exports timing information about when the replisome stalls and/or resumes.

2. stall_duration_bin_and_resumption_240s_stall.m
Script that collects timing information files generated by (1) and calculates statistics the stall/resumption fraction vs time.
